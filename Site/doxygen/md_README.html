Rocket

Ce code permet de calculer l'écoulement d'un gaz raréfié lors d'une rentrée atmosphérique

Fichiers

Le projet Rocket est constitué des fichiers suivants: -Readme.md, le fichier que vous êtes en train de lire -Un makefile qui permet de compiler Rocket sur Linux -Un fichier paramètres permettant de modifier les caractéristiques de l'écoulement -Un fichier .geo permettant de générer le maillage au format .mesh de l'INRIA -Un fichier .mesh contenant un maillage du domaine -L'ensemble des fichiers source

Compilez-le vous-même

Le compilateur g++ doit être préalablement installé.

Sous un système Linux, il est alors possible de compiler Rocket à partir du code source avec le Makefile inclus.

Ressource pour comprendre les bases du code

Le site du projet "ADRESSE DU SITE" explique en détails les modèles utilisés par Rocket.
